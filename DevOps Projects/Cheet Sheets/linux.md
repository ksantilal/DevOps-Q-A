# Linux Cheat Sheet

## ЁЯУБ File & Directory Commands
- `ls -la` тЖТ List all files (including hidden) with details
- `pwd` тЖТ Show current directory
- `cd /path` тЖТ Change directory
- `mkdir dir` тЖТ Create directory
- `rm -rf file/dir` тЖТ Delete file or directory recursively
- `cp src dest` тЖТ Copy files/directories
- `mv src dest` тЖТ Move or rename files

## ЁЯУЦ File Viewing
- `cat file` тЖТ Display file content
- `less file` тЖТ View file interactively (scrollable)
- `head -n 10 file` тЖТ First 10 lines
- `tail -f file` тЖТ Live file updates (logs)

## ЁЯФР Permissions
- `chmod +x file` тЖТ Make file executable
- `chown user:group file` тЖТ Change ownership

## ЁЯФН Search
- `find . -name file.txt` тЖТ Find file by name
- `grep "text" file` тЖТ Search text inside file

## ЁЯУК System Monitoring
- `top` тЖТ Real-time processes
- `htop` тЖТ Better UI version of top
- `df -h` тЖТ Disk usage
- `free -m` тЖТ Memory usage

## ЁЯМР Networking
- `ping host` тЖТ Check connectivity
- `curl url` тЖТ Fetch data from URL
- `wget url` тЖТ Download file
